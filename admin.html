📰 보도자료 추가

1. 기사 링크(URL)를 주소창에서 복사해 주세요.
2. 출처명은 언론사 이름입니다. (예: 도시정비뉴스, 마이고양)
3. 요약은 짧게 한두 줄이면 충분합니다. 생략 가능합니다.
4. 등록하면 홈페이지 보도자료 섹션에 바로 나타납니다.

🏠 홈페이지 바로가기

❓ 자주 막히는 상황 해결법

📱 사진이 안 올라갈 때
사진 파일이 너무 클 수 있습니다.
해결: 카카오톡으로 사진을 보낼 때 "원본"이 아닌 일반 화질로 저장하면 용량이 줄어듭니다.
또는 폰 갤러리에서 사진 선택 → 공유 → "크기 줄여서 저장" 옵션을 사용하세요.
🔄 등록했는데 홈페이지에 안 보일 때
1. 홈페이지를 완전히 새로고침 해주세요.
iOS: 주소창 왼쪽 새로고침(↺) 버튼 꾹 누르기
Android: 브라우저 메뉴 → 새로고침
2. 그래도 안 보이면 1~2분 기다렸다가 다시 확인해주세요.
🗑 실수로 삭제했을 때
삭제 전 확인창이 뜨기 때문에 실수로 삭제되는 경우는 드뭅니다.
만약 삭제되었다면 다시 등록해 주세요.
메시지 본문은 카카오톡·메모 앱에도 남아있을 수 있습니다.
🔐 비밀번호를 잊었을 때
현재 비밀번호: 담당자에게 문의하세요.
비밀번호는 홈페이지 소스코드에서 확인할 수 있습니다.
GitHub → index.html → ADMIN_PW 검색
📘 페이스북 링크 찾는 방법
1. 페이스북 앱에서 올린 글 찾기
2. 글 오른쪽 위 ··· 버튼 클릭
3. "게시물 링크 복사" 선택
4. 관리자 페이지 "페이스북" 탭에서 붙여넣기 → 저장

📞 기술 문의

홈페이지 관련 기술적인 문제가 생기면
담당자에게 화면 캡처와 함께 연락해 주세요.

꼭 캡처해야 할 것:
· 오류가 난 화면 전체
· 빨간색 오류 문구가 있으면 그것도 캡처
