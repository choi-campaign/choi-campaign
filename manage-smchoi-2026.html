✏️ 새 메시지 등록

💡 15자 내외가 가장 보기 좋습니다. 그날의 핵심 키워드를 넣어주세요.
예시: "사전투표 첫날, 여러분의 한 표를 부탁드립니다"
💡 200~400자가 읽기 가장 좋습니다. 너무 짧으면 성의 없어 보이고, 너무 길면 안 읽힙니다.
· 첫 문장에 핵심 메시지를 담으세요 (예: "오늘 사전투표를 했습니다")
· 문단은 2~3개로 나누고 줄바꿈을 활용하세요
· 마지막에 "감사합니다" 또는 해시태그로 마무리하면 좋습니다
· 이모지(😊🙏✊)를 적절히 넣으면 가독성이 높아집니다

📸 활동사진 등록

💡 10~20자가 적당합니다. 장소·활동 내용을 간결하게 적어주세요.
좋은 예: "중산1동 아파트 단지 인사", "일산시장 상인 간담회"
피할 것: "오늘 활동", "사진1" (너무 짧고 불명확)
📊 사진 장수 가이드
✅ 1~5장: 권장   ⚠️ 6~10장: 시간이 걸릴 수 있음   ❌ 10장 이상: 나눠서 올리세요

✅ 폰 원본 사진 그대로 올려도 됩니다.
자동으로 압축해서 저장하므로 화질 걱정 없이 올리시면 됩니다.
💡 대표 사진 3~5장만 올리고 나머지는 페이스북에 올리세요.

📰 보도자료 추가

1. 기사 링크(URL)를 주소창에서 복사해 주세요.
2. 출처명은 언론사 이름입니다. (예: 도시정비뉴스, 마이고양)
3. 요약은 짧게 한두 줄이면 충분합니다. 생략 가능합니다.
4. 등록하면 홈페이지 보도자료 섹션에 바로 나타납니다.
💡 기사 제목을 그대로 복사해서 붙여넣으세요. 직접 바꾸지 않아도 됩니다.
💡 50~80자가 적당합니다. 없어도 되지만 있으면 홈페이지에서 더 보기 좋습니다.
좋은 예: "최신명 후보가 인천2호선 연장 촉구 기자회견에 참석해 고양시 교통 개선을 촉구했습니다."

📘 페이스북 최신글 교체

1. 페이스북 앱에서 올린 글을 찾아주세요.
2. 글 오른쪽 위 ··· 버튼을 누르세요.
3. "게시물 링크 복사"를 선택하세요.
4. 복사한 링크를 아래 칸에 붙여넣고 저장하세요.
5. 홈페이지를 새로고침하면 바뀐 글이 보입니다.
💡 퍼가기 코드 전체 또는 링크 주소 모두 그냥 붙여넣으면 됩니다. 자동으로 인식합니다.

✅ 언제 업데이트하나요?
· 새 글을 올렸을 때 · 홈페이지에 다른 글을 보여주고 싶을 때
· "삭제됐거나 비공개" 오류가 표시될 때

⚠️ 중요: 글의 공개 범위가 "전체 공개"여야 홈페이지에서 보입니다.
"친구 공개"나 "나만 보기" 설정 글은 홈페이지에서 오류로 표시됩니다.

· 삭제된 메시지는 복구가 안 됩니다.
· 삭제 전 확인창이 뜨니 실수로 누르지 않도록 주의하세요.

🏠 홈페이지 바로가기

📋 홈페이지 메뉴 구성 안내

홈페이지는 아래 9개 섹션으로 구성되어 있습니다.
관리자 페이지에서 직접 수정 가능한 섹션은 ✅로 표시했습니다.
나머지는 담당자에게 수정을 요청해 주세요.
경력 (CANDIDATE STORY) — 담당자 수정
후보님의 경력·학력·활동 이력이 표시되는 섹션입니다.
내용 변경이 필요하면 담당자에게 요청해 주세요.
예: 새 직책 추가, 경력 수정 등
공약 (PLEDGE) — 담당자 수정
생활 교통 · 생활 돌봄 · 안심 환경 3개 탭으로 구성된 공약 섹션입니다.
공약 내용 추가·수정이 필요하면 담당자에게 요청해 주세요.
예: 새 공약 추가, 세부 내용 수정 등
✅ D-DAY 카운트다운 — 자동 작동
선거일(6월 3일)까지 남은 시간을 자동으로 카운트합니다.
별도 관리 불필요합니다. 선거 후에는 담당자에게 수정 요청해 주세요.
후원 (DONATION) — 담당자 수정
정치후원금 계좌번호가 표시되는 섹션입니다.
계좌번호나 후원 안내 내용 변경이 필요하면 담당자에게 요청해 주세요.
현재 계좌: 국민은행 291601-04-298864 (최신명후원회)
권역 (REGION) — 담당자 수정
권역별 세부 공약 이미지가 표시되는 섹션입니다.
이미지 교체가 필요하면 담당자에게 요청해 주세요.
포스터 (POSTER) — 담당자 수정
공약 홍보 포스터 5장이 표시되는 갤러리 섹션입니다.
새 포스터 추가나 교체가 필요하면 담당자에게 요청해 주세요.
✅ 현장 (ACTIVITY) — 관리자 페이지에서 직접 관리
선거운동 현장 사진·영상을 등록하는 섹션입니다.
이 관리자 페이지의 📸 활동사진 탭에서 직접 등록·삭제 가능합니다.
매일 활동 사진을 올려주세요.
✅ 메시지 (MESSAGE) — 관리자 페이지에서 직접 관리
후보자 메시지를 등록하는 섹션입니다.
이 관리자 페이지의 📋 메시지 탭에서 직접 등록·수정·삭제 가능합니다.
매일 또는 주요 일정마다 작성해 주세요.
✅ 보도 (PRESS) — 관리자 페이지에서 직접 관리
보도자료·기사 링크와 페이스북 최신글이 표시되는 섹션입니다.
이 관리자 페이지의 📰 보도자료 탭과 📘 페이스북 탭에서 직접 관리 가능합니다.
새 기사가 나오면 보도자료 탭에서 등록, 페이스북 글이 바뀌면 페이스북 탭에서 업데이트해 주세요.
✅ 응원하기 (SUPPORT) — 자동 수집, 관리자 페이지에서 삭제 가능
방문자가 남긴 응원메시지가 자동으로 표시되는 섹션입니다.
별도 등록 불필요하며, 부적절한 메시지만 이 관리자 페이지의 💬 응원메시지 탭에서 삭제하시면 됩니다.

❓ 자주 막히는 상황 해결법

📱 사진이 안 올라가거나 너무 오래 걸릴 때
사진 파일 용량이 너무 크면 업로드가 느리거나 실패할 수 있습니다.

해결 방법 1 (카카오톡)
카카오톡에서 사진을 저장할 때 "원본"이 아닌 일반 화질로 저장하세요.
카톡 채팅방에서 사진 꾹 누르기 → 저장 → 일반 화질 선택

해결 방법 2 (갤러리)
갤러리에서 사진 선택 → 공유 → "크기 줄여서 저장" 또는 "압축하여 공유"

해결 방법 3
사진을 한 장씩 나눠서 올려보세요. 한 번에 너무 많으면 느릴 수 있습니다.
🔄 등록했는데 홈페이지에 안 보일 때
등록은 됐지만 브라우저가 이전 화면을 보여주고 있는 경우입니다.

iPhone (사파리)
주소창 왼쪽 새로고침(↺) 버튼을 꾹 눌러서 "캐시 비우고 강력 새로고침" 선택

안드로이드 (크롬)
브라우저 오른쪽 위 점 세 개(⋮) → 새로고침

그래도 안 보이면
1~2분 기다렸다가 다시 확인해 주세요.
✏️ 메시지를 잘못 입력했을 때
수정 방법
메시지 탭 → 등록된 메시지 목록 → 수정하고 싶은 메시지의 "수정" 버튼 클릭
→ 위 입력폼에 기존 내용이 자동으로 채워짐
→ 내용 수정 후 "수정 저장" 버튼 클릭
🗑 실수로 삭제했을 때
삭제 버튼을 누르면 "삭제할까요?" 확인창이 먼저 뜨기 때문에 실수로 삭제되는 경우는 거의 없습니다.
만약 삭제되었다면 다시 등록해 주세요.

※ 삭제된 내용은 복구가 안 됩니다. 신중하게 눌러주세요.
🔐 비밀번호를 잊었을 때
비밀번호를 잊으셨다면 홈페이지 담당자에게 문의해 주세요.
카카오톡이나 전화로 문의하시면 안내해 드립니다.
🌐 홈페이지가 아예 안 열릴 때
인터넷 연결 상태를 먼저 확인해 주세요.
연결되어 있는데도 안 열리면 잠시 후 다시 시도하거나 담당자에게 알려주세요.

📞 기술 문의

홈페이지 관련 기술적인 문제가 생기면 담당자에게 화면 캡처와 함께 연락해 주세요.

꼭 캡처해야 할 것:
· 오류가 난 화면 전체
· 빨간색 오류 문구가 있으면 그것도 캡처
· 어떤 작업을 하다가 오류가 났는지 설명
